# 📚 Cours de C — Exercices de 3eme année de Licence science de la vie - parcours bio-informatique 

Bienvenue sur ce dépôt ! Il regroupe les exercices pratiques réalisés dans le cadre d'un cours d'introduction au langage C.

---

## 📂 Contenu du dépôt

| Thème | Description |
|---|---|
| `variables-types/` | Déclaration, initialisation et manipulation des types de base (`int`, `float`, `char`, etc.) |
| `pointeurs-memoire/` | Pointeurs, allocation dynamique, adresses mémoire |
| `fonctions/` | Définition, appel, passage par valeur et par référence |
| `tableaux-chaines/` | Tableaux statiques, chaînes de caractères, manipulation de strings |
| `structures/` | Définition de `struct`, imbrication, tableaux de structures |
| `fichiers/` | Lecture et écriture dans des fichiers (`fopen`, `fclose`, `fprintf`, `fscanf`, etc.) |

---

## 🛠️ Pré-requis

- Un compilateur C : [GCC](https://gcc.gnu.org/) (Linux/macOS) ou [MinGW](https://www.mingw-w64.org/) (Windows)
- Un éditeur de texte ou IDE (VS Code, gedite, CLion, Vim…)

---

## ▶️ Compiler et exécuter un exercice

```bash
gcc -Wall nom_du_fichier.c -o exercice
./exercice
```
